# 六
## 1.好的网页的标准?

1. 这是个什么样的网站
2. 网站上有什么
3. 我能在网站上做什么
4. 为什么我应该在这里
## 2.合格的网页是什么，具有哪些特点？

1. 美观舒适的风格
2. 高效的导航
3. 符合大众习惯的布局
4. 及时的反馈
5. 简介的操作
6. 明确的标识和指示
7. 拥有吸引热的元素
8. 良好的可维护性
## 3.（导航栏 ）是设计时最为用心的地方，把网页很好的连接起来，用户使用起来会更加方便.
